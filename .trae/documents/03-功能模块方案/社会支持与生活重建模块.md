# 社会支持与生活重建模块（合并文档）

> 本文档合并了社会支持模块的实施方案和收尾集成计划。

---

## 一、实施方案

### 概要

为"申归途"模块四"社会支持与生活重建"规划纯前端可落地功能。方案在 Resources 页面新增"🌱 重建"子 tab，包含 3 个功能区块：渐进式社交任务、同伴故事、社交活动记录。

**约束**：纯前端 React + LocalStorage，无后端，不新增路由，不改变 BottomNav。

### 可落地功能（3 项）

#### 1. 渐进式社交任务系统

基于行为激活理论，5 个阶段（微小社交→低强度→中等→挑战性→回归性），每阶段 3-4 个任务。用户自主选择节奏，标记完成，记录感受。

#### 2. 同伴故事展示

8 篇匿名化真实康复故事，按主题筛选（停药复发/社交恢复/职场回归/个人成长），卡片列表 + 详情阅读。

#### 3. 社交活动记录

快速记录每日社交活动（类型标签 + 备注），近 7/30 天频率统计。

### 不可实现功能

| 功能 | 原因 |
|------|------|
| 匿名社区 | 需后端存储+审核 |
| 同伴匹配 | 需后端用户数据库+匹配算法 |
| 引导小组 | 需后端多人实时通信 |
| 同伴培训 | 需后端认证管理 |

### 页面集成方案

Resources 页面新增第 6 个 tab "🌱 重建"。

```
现有：医院 | 热线 | 社区 | 医保 | 家属
新增：医院 | 热线 | 社区 | 医保 | 家属 | 重建
```

"重建" tab 内部用 `view` state 切换：社交任务 / 同伴故事 / 故事详情。

### 数据结构

#### `sgt-social-tasks`
```json
{ "micro_1": { "completedAt": "2026-05-01T10:30:00.000Z", "feeling": "没那么难" } }
```

#### `sgt-social-activities`
```json
[{ "date": "2026-05-01", "type": "chat", "note": "和朋友吃了午饭", "timestamp": 1746057600000 }]
```
