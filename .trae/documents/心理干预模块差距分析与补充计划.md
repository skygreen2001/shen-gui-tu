# 数字心理干预与技能训练 — 差距分析与补充计划

## 概要

设计方案要求 12+ 门三层递进课程（MBCT + CBT），当前仅实现 3 门纯 CBT 课程。本计划分析差距并规划可落地的补充方案。

---

## 一、设计要求 vs 当前实现

### 课程体系对比

| 层级 | 设计要求 | 当前实现 | 差距 |
|------|---------|---------|------|
| **第一层：基础技能** | 4门（认识情绪、身体扫描、自动思维识别、呼吸空间） | 2门（思维模式识别≈自动思维识别、认知重构≈认识情绪） | 缺身体扫描、呼吸空间 |
| **第二层：深化技能** | 5门（与低落共处、行为激活、认知灵活性、人际效能、正念自我关怀） | 1门（行为激活，内容较简略） | 缺4门 |
| **第三层：长期维护** | 4门（复发预防计划、预警信号手册、危机工具箱、意义重建） | 0门（WRAP页面独立存在但非课程形式） | 缺4门 |
| **合计** | **13门** | **3门** | **缺10门** |

### 框架能力对比

| 能力 | 设计要求 | 当前状态 |
|------|---------|---------|
| 渐进解锁 | ✅ | ✅ 已实现（prerequisites） |
| 微学习步骤制 | ✅ | ✅ 已实现（内容/练习/反思） |
| 练习重于理论 | ✅ | ✅ 已实现 |
| 灵活进度 | ✅ | ✅ 已实现（LocalStorage） |
| 多模态（音频/视频） | 文字+音频+视频 | ❌ 仅纯文本 |
| 文化适配（上海场景） | 中国/上海本地化 | ❌ 通用场景 |

## 二、差距优先级排序

按"纯前端可实现 + 用户价值"排序：

| 优先级 | 缺失内容 | 可行性 | 理由 |
|--------|---------|--------|------|
| **P0** | 补充 MBCT 正念课程（身体扫描、呼吸空间、与低落共处、正念自我关怀） | ✅ 纯文本即可 | 设计方案核心亮点，MBCT可降低5年复发率42% |
| **P0** | 补充第二层深化课程（认知灵活性、人际效能） | ✅ 纯文本即可 | CBT体系完整性 |
| **P1** | 呼吸空间引导音频（复用Crisis页4-7-8呼吸动画思路） | ✅ 可用定时器+文字引导 | 多模态入门，难度低 |
| **P1** | 第三层课程（复发预防计划、预警手册、危机工具箱） | ⚠️ 与WRAP页面功能重叠 | 可整合为"WRAP进阶课程"引导用户完善WRAP |
| **P2** | 意义重建与价值探索（ACT模块） | ✅ 纯文本即可 | 第三层最独特的课程 |
| **P2** | 文化适配（上海本地化案例） | ✅ 修改现有课程文本 | 提升代入感 |

## 三、实施方案

### 方案概述

在现有 `cbtContent.js` 中补充课程数据，复用 `CBTCourse.jsx` 组件（无需新建页面），按三层重新组织课程列表。

### Step 1：重构课程数据结构

**文件**：`src/data/cbtContent.js`

为课程增加 `tier`（层级）和 `tag`（标签）字段，支持分层展示：

```js
// 新增字段
{ id: 1, title: '...', tier: 1, tag: 'CBT', ... }
{ id: 4, title: '身体扫描入门', tier: 1, tag: 'MBCT', ... }
```

### Step 2：补充第一层课程（+2门）

| ID | 课程名 | 标签 | 步骤数 | 前置 |
|----|--------|------|--------|------|
| 4 | 身体扫描入门 | MBCT | 4步 | 无 |
| 5 | 三分钟呼吸空间 | MBCT | 3步 | 无 |

### Step 3：补充第二层课程（+4门）

| ID | 课程名 | 标签 | 步骤数 | 前置 |
|----|--------|------|--------|------|
| 6 | 与低落情绪共处 | MBCT | 5步 | 完成课程1 |
| 7 | 认知灵活性训练 | CBT | 4步 | 完成课程2 |
| 8 | 人际效能提升 | CBT | 5步 | 完成课程1,2 |
| 9 | 正念自我关怀 | MBCT | 4步 | 完成课程4或5 |

### Step 4：补充第三层课程（+3门）

| ID | 课程名 | 标签 | 步骤数 | 前置 |
|----|--------|------|--------|------|
| 10 | 我的复发预防计划 | WRAP | 4步 | 完成课程3 |
| 11 | 预警信号手册 | WRAP | 3步 | 完成课程10 |
| 12 | 意义重建与价值探索 | ACT | 5步 | 完成课程6,9 |

> 注：第三层"危机自救工具箱"与 Crisis 页面和 WRAP 危机计划高度重叠，不再单独设课，避免冗余。

### Step 5：修改 CBTCourse.jsx 支持分层展示

**文件**：`src/pages/CBTCourse/CBTCourse.jsx`

- 课程列表按 tier 分组显示（基础技能 / 深化技能 / 长期维护）
- 每组有标题和简要说明
- 课程卡片增加 tag 标签（CBT / MBCT / WRAP / ACT）
- 其余交互逻辑不变（渐进解锁、步骤导航、进度保存）

### Step 6：构建验证

`npx vite build` 确认 0 错误。

## 五、不做的事项

| 项目 | 原因 |
|------|------|
| 音频/视频内容 | 需要实际录制或外部资源，纯前端无法生成高质量音频 |
| 新建独立页面 | 复用现有 CBTCourse 组件即可，不增加路由复杂度 |
| ACT 深度课程 | 仅保留1门入门级，ACT 需专业引导不适合纯自助 |
| 文化适配全面重写 | 工作量大，新课程直接使用上海场景即可，旧课程暂不改 |
