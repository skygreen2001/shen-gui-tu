> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在2小时内使用 Trae MTC 完成一个可离线运行的 React SPA 前端原型——「申归途」上海抑郁症复发预防支持工具

**Architecture:** Vite + React 18 + HashRouter（离线兼容）+ CSS Modules + LocalStorage 数据持久化。移动端优先（max-width: 480px），零外部运行时依赖，纯前端无后端。

## 关键设计决策

1. **HashRouter**：离线文件协议兼容
2. **CSS Modules**：零外部依赖，组件样式隔离
3. **Emoji图标**：无需图标库，离线可用
4. **纯CSS图表**：不引入ECharts等重依赖
5. **LocalStorage**：简单可靠的无后端数据方案
6. **移动端优先**：max-width: 480px 居中布局
