# 家庭/照护者支持模块 — 纯前端实施方案

## 概要

为"申归途"模块五"家庭/照护者支持"规划纯前端可落地功能。方案在 Resources 页面新增"家属"子 tab，包含 4 个功能区块：教育课程、行动指南、压力自评、自我关怀清单。

**约束**：纯前端 React + LocalStorage，无后端，不新增路由，不改变 BottomNav。

---

## 一、可落地功能（4 项）

### 1. 家属心理教育课程（6 门）

复用 CBT 课程模式（列表 → 步骤浏览 → 进度存储）。

| 课程 | icon | 内容 |
|------|------|------|
| 理解抑郁症 | 🩺 | 科学认识、神经生物学、常见误区 |
| 识别复发信号 | 🔍 | 六维预警家属版、观察技巧 |
| 有效沟通技巧 | 💬 | 积极倾听、共情回应、实用话术 |
| 支持边界 | 🚧 | 支持不控制、避免过度保护 |
| 自我关怀 | 🌿 | 照顾者倦怠、情绪管理、寻求帮助 |
| 危机应对 | 🆘 | 风险识别、紧急步骤、求助渠道 |

**入口**：Resources → "👨‍👩‍👧 家属" tab → 课程卡片
**进度**：`useLocalStorage('sgt-family-progress', {})`

### 2. 差异化家属行动指南（四级风险）

复用现有 green/yellow/orange/red 四级体系，每级包含：状态描述、可做的事、应避免的事、关键提醒。

**入口**：Resources → "👨‍👩‍👧 家属" tab → 行动指南区块

### 3. 照护者压力自评（8 题）

Likert 5 级量表，8 题，总分 8-40 分，四级结果解读。

**入口**：Resources → "👨‍👩‍👧 家属" tab → "开始自评"按钮
**记录**：`useLocalStorage('sgt-caregiver-assessments', [])`，限 30 条

### 4. 自我关怀清单

参照 WRAP 的可编辑条目列表，预置关怀建议，可自定义。

**入口**：Resources → "👨‍👩‍👧 家属" tab → 折叠面板
**数据**：`useLocalStorage('sgt-caregiver-selfcare', defaultItems)`

---

## 二、不可实现功能

| 功能 | 原因 |
|------|------|
| 家属独立账户 | 需后端认证+多角色 |
| 匿名支持社区 | 需后端存储+审核 |
| 信息推送 | 需后端定时任务 |
| 知情同意/信息分级 | 需后端多用户绑定 |

---

## 三、页面集成方案

**方案**：Resources 页面扩展，新增第 5 个子 tab

```jsx
// Resources.jsx tabs 数组
const tabs = [
  { key: 'hospitals', label: '🏥 医院' },
  { key: 'hotlines', label: '📞 热线' },
  { key: 'community', label: '🏘️ 社区' },
  { key: 'insurance', label: '📋 医保' },
  { key: 'family', label: '👨‍👩‍👧 家属' },  // 新增
]
```

"家属" tab 内部布局（从上到下）：
1. 📖 家属教育课程（6 张卡片网格）
2. 🎯 家属行动指南（4 级颜色编码卡片）
3. 💪 照护者自评（按钮 + 历史记录）
4. 🌿 自我关怀清单（可编辑列表）

课程详情用 `useState` 内联视图切换（同 CBTCourse 模式），不新增路由。
